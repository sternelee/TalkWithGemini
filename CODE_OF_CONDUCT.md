# Code of Conduct

Neo Chat is an open-source project for people with different backgrounds,
languages, and levels of experience. We expect contributors and maintainers to
keep collaboration respectful, focused, and safe.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, discussions, and
  review comments.
- Assume good intent, but ask for clarification when something is unclear.
- Keep technical disagreements focused on evidence, tradeoffs, and project
  goals.
- Respect user privacy. Do not post API keys, access passwords, private chat
  logs, or other sensitive data.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Sexualized language or unwanted personal attention.
- Publishing another person's private information without explicit permission.
- Repeated disruption of project spaces after maintainers ask for a change in
  behavior.

## Reporting

For conduct concerns, open a private security advisory if the report includes
private or sensitive information, or contact the maintainers through the GitHub
repository owner. Maintainers will review reports and may remove content,
moderate participation, or block contributors when needed to protect the
project community.

## Scope

This code applies to project repositories, issue trackers, pull requests,
community spaces, and public representation of the project.
